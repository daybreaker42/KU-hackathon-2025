# 식물 관리 앱 API 기능 목록

일단은 대분류 위에 있는 것부터 만들기.

## 1. 홈 페이지 (Home)
1.1. 내 식물 리스트 가져오기
1.1.1 식물별 상태도
1.2. 오늘 할 일 가져오기 (이거 조금 더 생각 필요, 오늘 급수+햇빛 일정 있는 식물들만 filter해서 count()?)
